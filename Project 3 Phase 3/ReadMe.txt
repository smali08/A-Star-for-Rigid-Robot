A-star Algorithm for Rigid Differential Drive Robot
Authors - Shesh Mali and Kumar Sambhav Mahipal

Dependencies

*Numpy
-Open the command prompt and type:
pip install numpy

*Matplotlib
-Open the command prompt and type:
pip install matplotlib


Running from zip - 

- Download zip folder proj3_33_gazebo_python.zip
- Extract the zip folder
- Open command prompt

cd (directory where zip is extracted)
cd proj3_33_gazebo_python
cd Phase_3
python3 Phase3.py

After the code execution begins
->Enter the Clearance of the robot in mm
->Enter the RPM of the Robot (RPM1,RPM2) -> RPM1 RPM2
->Enter the co-ordinates of starting point separated by space in mm (x,y,theta_s) --> x y theta_s
->Enter the co-ordinates of goal point separated by space in mm  (x,y) --> x y 
Hit Enter

Submission Video Parameters:
-> Clearance -> 50
-> RPM -> 50 100
-> Start -> -4600 -4600 0
-> Goal -> 4600 4600
